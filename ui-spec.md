# UI/UX Specification

## Design Direction

**Tone:** Festive but sophisticated. Warm and inviting without being kitschy or overwhelming.

**Color Palette:**
- Primary: Deep green (#1a472a) or rich burgundy (#722f37)
- Accent: Gold/champagne (#d4af37)
- Background: Warm off-white (#faf9f6) or soft cream
- Text: Dark charcoal (#2d2d2d)
- Success: Forest green (#228b22)
- Cards: White with subtle shadow

**Typography:**
- Headers: A warm serif or elegant sans-serif (e.g., Playfair Display, Crimson Pro)
- Body: Clean readable sans-serif (e.g., Source Sans Pro, Lato)
- Numbers/amounts: Slightly bolder weight for emphasis

## Animation Specifications

### Card Flip Animation (Framer Motion)

```typescript
// CardFlip.tsx
const cardVariants = {
  unflipped: {
    rotateY: 0,
    scale: 1
  },
  flipped: {
    rotateY: 180,
    scale: 1.1,
    transition: {
      duration: 0.6,
      ease: [0.4, 0, 0.2, 1]
    }
  }
};

const frontVariants = {
  visible: { opacity: 1, rotateY: 0 },
  hidden: { opacity: 0, rotateY: 180 }
};

const backVariants = {
  visible: { opacity: 1, rotateY: 0 },
  hidden: { opacity: 0, rotateY: -180 }
};
```

### Modal Zoom Animation

```typescript
// DayCardModal.tsx
const modalVariants = {
  hidden: {
    opacity: 0,
    scale: 0.8,
    y: 20

  visible: {
    opacity: 1,
    scale: 1,
    y: 0,

      stiffness: 300,
      damping: 25
    }
  },
  exit: {
    opacity: 0,
    scale: 0.8,
    y: 20,
    transition: {
      duration: 0.2
    }
  }
};
```

### Grand Prize Confetti

Use a library like `react-confetti` or `canvas-confetti` for the grand prize reveal.

```typescript
// On grand prize reveal
import confetti from 'canvas-confetti';

function triggerGrandPrizeConfetti() {
  confetti({
    particleCount: 100,
    spread: 70,
    origin: { y: 0.6 },
    colors: ['#d4af37', '#1a472a', '#722f37'] // gold, green, burgundy
  });
}
```

## Responsive Breakpoints

```css
/* Mobile first */
.calendar-grid {
  display: grid;
  grid-template-columns: repeat(7, 1fr);
  gap: 0.5rem;
}

/* Small screens: smaller cards */
@media (max-width: 640px) {
  .calendar-grid {
    gap: 0.25rem;
  }
  .day-card {
    font-size: 0.75rem;
    padding: 0.5rem;
  }
}

/* Large screens: more breathing room */
@media (min-width: 1024px) {
  .calendar-grid {
    gap: 1rem;
  }
  .day-card {
    padding: 1rem;
  }
}
```

## Accessibility Considerations

1. **Keyboard navigation**: Cards focusable with Tab, activated with Enter/Space
2. **Screen readers**: Announce card state (unrevealed, revealed, paid)
3. **Color contrast**: Ensure text meets WCAG AA standards
4. **Focus indicators**: Visible focus rings on interactive elements
5. **Motion preferences**: Respect `prefers-reduced-motion` for animations
6. **Alt text**: Descriptive labels for all icons and images
